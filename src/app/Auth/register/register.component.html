Data Playground(DP) is a Full Stack Web Application, that runs on Django and Django Rest Framework for the Backend and REACT and Redux for the Frontend. With the DP dashbord you can upload CSV files to train Machine Learning Models such as Linear Regression and Logistic Regression. These models will then be saved on the application's database so that you can run them any time either through the Front End or through the REST Api.
